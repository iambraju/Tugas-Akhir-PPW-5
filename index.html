Penjelasan tombol:
  • MC — Memory Clear: kosongkan memory
  • MR — Memory Recall: ambil nilai memory ke layar
  • M+ — Memory Add: tambahkan nilai layar ke memory
  • M- — Memory Subtract: kurangi nilai layar dari memory
  • CE — Clear Entry: hapus angka yang sedang diketik
  • C — Clear All: hapus seluruh yang ada (ekspresi + entri)
