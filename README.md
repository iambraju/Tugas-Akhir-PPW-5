# Interactive Calculator 


## File
- `index.html` — markup UI
- `style.css` — styling responsive
- `script.js` — seluruh logika kalkulator

## Fitur
- Basic arithmetic: +, -, ×, ÷
- Chain calculations with precedence
- CE (Clear Entry) & C (Clear All)
- Decimal point
- Keyboard support
- Division-by-zero handling and invalid expression handling
- History (last 5) stored in localStorage
- Memory: M+, M-, MR, MC (persist in localStorage)

## Menjalankan
1. Simpan file `index.html`, `style.css`, `script.js`,  ke satu folder.
2. Buka `index.html` di browser.
